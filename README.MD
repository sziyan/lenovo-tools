## Lenovo Warranty 

This Telegram bot scrapes Lenovo warranty site to extract warranty end date.

Currently supports the following laptop models:

- Thinkpad E14
- Thinkpad E490
- Thinkpad L13
- Thinkpad L380
- Thinkpad L390
- Thinkpad L480
- Thinkpad L490
- Thinkpad X280
- Thinkpad X390
- Thinkpad L14
- Thinkpad X13
